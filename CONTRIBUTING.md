# Contributing to GreenCalculus Methodology

Thanks for your interest! This repository holds the formal methodology, emission factor
datasets, and GHG Protocol alignment documentation behind GreenCalculus. It is open source
(MIT) and we welcome corrections and improvements from the community.

## Ways to contribute

- **Correct a factor** — if an emission factor is outdated, mis-sourced, or wrong, open an
  [issue](../../issues) or PR with the corrected value **and a link to the authoritative
  source** (GHG Protocol, IPCC AR6, DEFRA, IEA/Ember, national inventory, etc.).
- **Add a source** — point us to a published dataset or standard we should incorporate.
- **Clarify methodology** — flag anything ambiguous in a formula, boundary, or allocation choice.
- **Documentation** — fix typos, broken links, or unclear explanations.

## Submitting a pull request

1. Fork and create a branch.
2. Keep changes focused and cite every factor or methodology change.
3. Describe **what** changed, **why**, and **the source**.
4. Open the PR against `main`.

## Ground rules

- Every emission factor must be traceable to a published, authoritative source — include the
  citation (publisher, document, year, table/page) in the change.
- Be respectful and constructive.

Questions? Open an issue or reach us at jeremiah@greencalculus.com.
